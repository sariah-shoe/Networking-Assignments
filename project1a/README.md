To run the program: 
    Run "python client.py server.example.com port#"
    To play against class server, set compnet.cs.du.edu as the server and set 7575 as the port

Once the program has begun:
    The server will be contacted and choose Xs or Os
    If the server is X, it will make the first move, otherwise you will
    You will be prompted to make your move in the format of x,y
    The game continues until there is a winner or a tie

To play again, run the program again
